# LFAS Portfolio Roadmap

## MVP — Statement Intelligence

Outcome:

```text
Upload statement
→ Extract transactions
→ Normalize and validate
→ Categorize transactions
→ Generate financial report
```

## MDP — Financial Decision Platform

Outcome:

```text
Statement analysis
+ Questionnaire
→ Scores
→ Recommendations
→ Financial decision guidance
```

## V1 — Commercial SaaS Platform

Outcome:

```text
Multi-tenant commercial platform
+ Billing
+ AI assistance
+ Open banking readiness
+ Forecasting and risk analytics
```

## Guiding Rule

AI is an enhancement layer. Deterministic parsing, validation, scoring, and recommendation rules remain the source of truth.
